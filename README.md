# pi1-univesp

Sistema web de controle de estoque de autopeças, desenvolvido como Projeto Integrador em Computação I da UNIVESP.

## Tecnologias
- PHP
- JavaScript
- MySQL

## Objetivo
Permitir cadastro de peças, controle de estoque e associação de compatibilidade com veículos.

## Estrutura inicial
- `index.php`: página inicial
- `app/config/conexao.php`: conexão com banco
- `database/schema.sql`: estrutura inicial do banco
